Project Overview

Voice spoofing is a growing cybersecurity threat where attackers use AI-generated voices to impersonate real people. This project focuses on detecting such spoofed calls using machine learning and signal processing techniques.


Objectives

  • Detect AI-generated and cloned voices
  • Prevent voice-based fraud
  • Improve call authentication security
